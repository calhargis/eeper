# fixtures — labeled audio fixture library (M2.0)

CI-only tooling that produces a frozen, versioned library (`fixtures-v1`) of
synthesized nursery scenes for the M2.3 cry-detection quality gate — built from
`manifest.json` alone, with **no third-party audio committed** to the repo. This is
a long-lead milestone that needs only CI (not the running stack).

## What it does

- **`manifest.json`** pins every source clip: a stable fetch reference, a SHA-256, a
  license from the allowed (non-NC) set, labels, category, split, and a human
  verification status.
- **`fixtures build`** fetches + checksum-verifies each source clip (refusing any
  tampered/substituted file), then synthesizes scenes with **Scaper** — a cry or
  confuser event placed over the nursery noise floor at a swept SNR with light room
  reverb — **deterministically**, so two clean builds are byte-identical.

## CLI

```
fixtures verify                 # manifest integrity + allowed-license + source-split disjointness
fixtures build --out DIR        # fetch/generate sources and synthesize scenes (+ scenes.json)
fixtures check --out DIR        # statistical floor + annotation sanity + scene-split gates
fixtures repro --a A --b B      # assert two built libraries are bit-identical
fixtures provenance --out FILE  # render the license/provenance report
```

## Sources (license-clean; NC excluded by policy)

| Bucket | Source | License |
| --- | --- | --- |
| cry positives | [donateacry-corpus](https://github.com/gveres/donateacry-corpus) (pinned commit) | ODbL-1.0 |
| speech / music-TV / pets | FSD50K via the `sumin0223` clip mirror (pinned); per-clip license from the Fhrozen metadata | CC0-1.0 / CC-BY-3.0 (by-nc + sampling+ dropped) |
| white-noise / lullaby / nursery floor | generated (deterministic) | CC0-1.0 |

`ESC-50` and any NC-licensed source are excluded by policy; the validator enforces
the allow-list. See [`PROVENANCE.md`](PROVENANCE.md) (generated by `fixtures provenance`).

## `fixtures-v1` scope

Confuser categories: **speech, music_tv, pets, whitenoise_lullaby** — ≥ 30 eval
scenes each, ≥ 300 confuser total, ≥ 100 cry. The **sibling/other-child** category
and richer pets-dev coverage are deferred to **fixtures-v1.1**, pending
project-recorded child-speech clips (there is no clean fetchable source — Common
Voice is gated, FSD50K child-speech is sparse). The tooling already ingests
`eeper-recorded` clips, so adding them is a manifest change + a fixture-version bump.

## Reproducibility

The build runs in a **digest-pinned base image** with a pinned `requirements.lock`,
so the audio backend (numpy/scipy/soundfile/soxbindings + SoX/ffmpeg/libsndfile) is
fixed — the precondition for bit-identical output. CI (`.github/workflows/fixtures.yml`,
x86_64) runs two clean builds and asserts byte-identity. Any change to a source clip,
the generator, or the audio backend bumps the fixture version and re-baselines the
M2.3 gate (recorded in `docs/PROGRESS.md`).

## Regenerate the manifest

```
python tools/build_manifest.py    # occasional; fetches + hashes the pinned sources
```
